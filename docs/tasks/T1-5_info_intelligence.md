# T1-5 信息筛选（T3 团队 · P1 版）

**目标**：抓取行业/市场信息，去重、分级、关联资产，写入飞书 `intel` 表，产出每日简报。
**要做什么**
- 数据源 P1 先用 AkShare 新闻接口 / RSS（可复用 `data_fetcher.get_news`，或新建 `core/intel.py`）。
- 处理：去重 → 摘要 → 重要性分级（必读/可读/忽略）→ 关联资产/行业标注 → 每条标来源与时间。
- 需要 AI 判断重要性/摘要时，**统一用 `core/llm_client.chat(...)`**（见 docs/README「LLM 调用」），不要自写 requests。
- 写入 `intel` 表（P0 已建占位表，必要时补字段）。
- （可选）Streamlit 加「信息流」页：浏览/收藏/忽略/转研究任务。
**纪律**：先定义「什么值得收」再收；不把未验证传闻当事实；不追热点制造噪音。
**交付/验收**：能产出已分类、已去重的简报并入库；每条带来源/时间/重要性；满足 Global DoD。
**依赖**：T1-1（若复用取数）。
