# T0-8 质量审查角色 + 示范个股研究

**目标**：① 确认 `agents/quality_review.md` 角色可用；② 产出一份示范个股研究存入飞书，让用户看到深度产出形态。
**要做什么**
- 选一只用户关注的 A 股，按 `templates/research_memo.md` 做研究，并由质量审查角色补"反方观点 + 风险审查 + 证伪条件"。
- 研究须满足质检：关键事实带来源、有反方、有不确定性标注、结论非买卖指令。
- 存入飞书（研究报告库或 Wiki 页），并在 tasks 表登记。
**交付/验收**（FR-R, FR-X）：一份完整示范研究可读；含反方与风险与证伪条件。
**依赖**：T0-2（存放位置）。可与其他任务并行。
